# 📱 Job Recruitment Android Application

## 📌 Project Overview
The Job Recruitment Application is an Android-based mobile application developed to simplify the job search and recruitment process. The application allows administrators to post job listings and enables users to browse available jobs and apply directly through the app.

This project demonstrates the implementation of mobile application development concepts such as database integration, user interaction, and dynamic job listing management.

---

## 🎯 Objectives

- To develop a mobile-based job recruitment system  
- To allow administrators to post and manage job listings  
- To enable users to view available jobs  
- To allow users to apply for jobs  
- To manage job data using a local SQLite database  

---

## ✨ Features

### 👤 User Features
- View available job listings  
- Browse job details  
- Apply for jobs  
- Simple and user-friendly interface  

### 🛠️ Admin Features
- Add new job postings  
- Delete job listings  
- Manage available job positions

## 🚀 How to Run the Project

1. Open **Android Studio**
2. Click **Open Project**
3. Select the project folder:

MADAssignment_JobRecruitment

4. Wait for Gradle sync to finish
5. Connect Android device or start emulator
6. Click **Run ▶️**
